	Vector, ArrayList의 함수들의 10개 정도 의미와 사용법 공부
	
	<ArrayList>
		java.util -> Classes ArrayList
	
	add(e) 
		- 순차적으로 데이터 넣기
	add(int index, element) 
		- 특정 인덱스에 데이터 넣기
	clear() 
		- 리스트의 모든 데이터 삭제

		- 인덱스의 데이터 꺼내기
	indexOf(Object o)
		- 인덱스 중 o 의 데이터가 첫 번째로 오는 위치값 반환. 없으면 -1 반환
